## 📖 About

TCGStore is a full-stack e-commerce platform for buying and selling trading card game (TCG) cards from popular games like: 

- 🎴 Pokémon
- 🃏 Magic: The Gathering  
- 🎲 Yu-Gi-Oh! 

---

## 🛠️ Tech Stack

### Backend
- **Java 17**
- **Spring Boot 3.x**
- **Spring Data JPA**
- **MySQL 8.0**
- **Maven**

### Frontend
- **React 18**

## 🚀 Getting Started

### Prerequisites

- Java 17+
- Node.js 16+
- MySQL 8.0+
- Maven 3.6+

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
